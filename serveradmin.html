Useful Server Commands:

Hotkeys:

' (tilde): Opens the server console.

SHIFT + P: Enables AdminCam.

1 + scroll wheel: Changes your speed while in AdminCam.

2 + scroll wheel: Zoom in and out while in AdminCam.

6 : Display the capture zones around each objective in the map while in AdminCam.

9 : Display the player nametags in the map while in AdminCam.

0 : Display the Admin HUD (located in top-left of screen) while in AdminCam.


General Server Commands:

AdminForceAllRoleAvailability [0 / 1]: Allows players to choose whichever kit they want.

AdminNoTeamChangeTimer [0 / 1]: Disable Team change timer.

AdminNoRespawnTimer [0 / 1]: Disable Respawn timer.

DebugToggleGodMode: Toggles GodMode.

AdminBroadcast [Message]: Send a message to the entire server. (the yellow messages that shows up in the top center of the screen)

AdminWarn [NameOrSteamId] [WarnReason]: Send a warning message to the specified player.

Server Time/Layer Commands:

AdminEndMatch: Ends the match.

AdminRestartMatch: Restarts the match.

AdminPauseMatch: Pauses the match.

AdminUnpauseMatch: Unpauses the match.

AdminSlomo [TimeDilation]: Sets the server speed to the specified multiplier. 1 is normal. .5 is half-speed. 2 is double speed.

AdminChangeLayer [LayerName]: Changes the layer to the specific map/layer.

AdminSetNextLayer [LayerName]: Sets the next layer to the specified map/layer.

Player-related Commands:

AdminDemoteCommander [PlayerName]: Demotes the specified player from the Commander position.

AdminDisbandSquad [TeamNumber] [SquadIndex]: Disbands the specified squad from the specified team. While in scoreboard: Left team is 1, Right team is 2

AdminRemovePlayerFromSquad [PlayerName]: Removes the specified player from their squad.

AdminForceTeamChange [NameOrSteamId]: Forces the specified player to the opposite team.

AdminKick [NameOrSteamId] [KickReason]: Kicks a player with a reason.

AdminBan [NameOrSteamId] [BanLength] [BanReason]: Bans a player for a specified time length with a reason. 0 = Perm, 1d = 1 Day, 1M = 1 Month. Example : Adminban Hodor 7d Attacking main base

AdminTeleportToPlayer [PlayerName]: Teleports to the specified player.

Deployable Commands:

AdminAlwaysValidPlacement [0 / 1]: Allows deployables to be placed anywhere without any restrictions whatsoever.

AdminForceAllDeployableAvailability [0 / 1]: Removes restrictions required to place down deployables.

AdminCreateDeployable [DeployableLink]: Spawns in specified deployable at the admins location.

Vehicle Commands:

AdminCreateVehicle [VehicleLink]: Spawns in specified vehicle at the admins location.

AdminDisableVehicleKitRequirement [0 / 1]: Allows players to access vehicles that have kit requirements (Pilot / Crewman vehicles)

AdminDisableVehicleTeamRequirement [0 / 1]: Allows players to access any vehicle, including enemy vehicles.

AdminDisableVehicleClaiming [0 / 1]: Allows players to claim a vehicle without SL permission.
